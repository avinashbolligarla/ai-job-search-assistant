# 🤖 AI Job Search Assistant

An AI-powered resume analyzer that matches your resume to any job description using NLP and Machine Learning.

Built by **Avinash Bolligarla** — MS IT Management, Webster University, San Antiono TX

---

## 🚀 What it does

- Paste your resume and a job description
- Instantly get a **Match Score** and **ATS Score**
- See which skills you have vs which ones are missing
- Get personalized recommendations to improve your resume

---

## 🛠️ Tech Stack

- **Python** — Core language
- **Flask** — Web framework
- **Scikit-learn** — TF-IDF vectorization & cosine similarity
- **Pandas & NumPy** — Data processing
- **HTML/CSS/JavaScript** — Frontend UI

---

## ⚙️ How to run locally

```bash
# Clone the repo
git clone https://github.com/avinashbolligarla/ai-job-search-assistant.git
cd ai-job-search-assistant

# Create virtual environment
python3 -m venv venv
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Run the app
python3 app.py
```

Open your browser and go to: `http://127.0.0.1:5000`
